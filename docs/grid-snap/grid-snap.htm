Snap to Grid

Introduction

There are basically three steps:

  1. Transform point coordinates to grid coordinates
  2. Divide coordinates by grid width, round, then multiply by grid width
  3. Transform grid coordiates back to point space

In a cartesian grid system,

Snapping line ends to the grid: A secondary pointer is shown in addition to the (mouse) pointer. The secondary pointer snaps to the grid. The user can move the primary pointer to select a propert line starting point. A mousedown event removes the secondary pointer and creates a line element with the first end at the position of the secondary pointer. Mousemove events update the line end position and snap it to the grid.
