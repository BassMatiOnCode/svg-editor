SVG Canvas Element

Sizing, zooming, pannings

Introduction

The size of the SVG viewport is defined by its viewBox property with its components x, y, w and h. The SVG viewport is scaled to fit the element width (defined by the SVG element's width attribute). With a zoom factor of 1.0, viewBox.w equals the SVG element's width. For a zoom factor greater than 1, the width is scaled up. For a zoom factor smaller than 1, the width is scaled down accordingly. Note that the height property is left undefined, which ensures that the aspect ratio is kept at 1.0.

While zooming, the viewport origin (0, 0) remains in its position, objects at a distance from the origin are move towards or away from the origin. It is desirable that a user-defined "focus" point remains fixed in position instead. This can be achieved by panning the document by a suitable amount, if possible. This is usually the case when zooming into the drawing.
